Welcome to the source code repository for my personal portfolio website. This project showcases my work, skills, and experience as a **Python Full-Stack Developer & Data Analyst**. 

The portfolio is built as a highly responsive, modern web experience with a premium, high-contrast **Dark Navy Blue & Glassmorphism** aesthetic.

---

## 🚀 Live Preview / Quick Start

Since this is a responsive static website, you can easily run and preview it locally on your machine.

### Prerequisites
Make sure you have [Python](https://www.python.org/) installed (comes by default with most developer setups).

### Run Locally
1. Clone or download this repository.
2. Open your terminal/command prompt in the project root directory.
3. Start a local development server using Python:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```
